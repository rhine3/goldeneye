About Goldeneye

Goldeneye is a quick reference for North American bird species' alpha codes and common names. It is also available as a Chrome extension.

Goldeneye draws from two alpha code authorities:
  • The Bird Banding Laboratory (BBL), and
  • The Institute for Bird Populations (IBP).
Some species have codes that differ between authorities. Some species are only included by one authority.
